Projects

Listed below are a couple of personal projects that I have worked on in my free time. I have yet to work on any projects that I would consider noteworthy for any of my Computer Science courses so far. It is my hope that future classes will provide me with projects that are much more interesting, which I then can add here.

  • Cracker Barrel Peg Game

    Developed using the Pygame Zero game engine on the MU python editor, this project simulates the peg game that one would find at Cracker Barrel Restauraunts. The goal of the game is to eliminate all but one peg from the board by having the pegs jump over one another. Are you an 'Eg-no-ra-moose' or a 'Genius'?

  • 3-D Tic-Tac-Toe

    Developed using the Pygame Zero game engine on the MU python editor, this project simulates a game of 3-dimensional tic-tac-toe. With the added z-axis, there are a total of 49 different ways that either player can win.
